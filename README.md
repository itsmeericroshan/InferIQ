# Fortress Compute — InferIQ
### *Next-Generation AI Inference Optimization Platform*

This repository contains the source code for the **InferIQ** platform, a high-performance interface designed for monitoring, managing, and optimizing AI model inference across decentralized compute networks.

---

## Key Features

- **Real-time Monitoring**: Visual feedback on inference speed, latency, and throughput.
- **Decentralized Compute Integration**: Seamless orchestration across Fortress Compute nodes.
- **Optimization Suite**: Built-in tools for pruning, quantization, and model compression visualization.
- **Premium UI/UX**: Designed for technical ops teams with high-fidelity data visualization.

---

## Tech Stack

- **Frontend**: React.js / HTML5 
- **Styling**: Vanilla CSS (Custom Variable System)
- **Data Viz**: Integrated performance charts and real-time telemetry simulations.

---

## Quick Start & Local Development

To run this project locally and explore the interface:

1.  **Clone the repository** (or download and unzip).
2.  Navigate into the project directory:
    ```bash
    cd fortress-compute
    ```
3.  **Install dependencies**:
    ```bash
    npm install
    ```
4.  **Launch the development server**:
    ```bash
    npm start
    ```
5. Open [http://localhost:3000](http://localhost:3000) in your browser.

> [!NOTE]
> You can also view the application by opening `index.html` directly in most modern browsers, though a local server is recommended for the best experience with dynamic assets.
