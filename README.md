# Fortress Compute - InferIQ

This is the extracted source code for the InferIQ AI Inference Optimization Platform.

## Local Development

To view the site locally:

1.  Navigate into the `fortress-compute` folder.
2.  Run `npm install` (optional, for the preview server).
3.  Run `npm start` to launch a local server.
4.  Open `http://localhost:3000` in your browser.

Alternatively, you can just open `index.html` directly in a browser, though some assets might require a local server to load correctly due to CORS or relative path handling in some browsers.

## Project Structure

-   `index.html`: Main entry point.
-   `static/js/bundle.js`: Main React application logic.
-   `assets/`: Shared scripts and images.
-   `fonts/`: Localized font files.
